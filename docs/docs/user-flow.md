#User Flow – H.MTALII (THE SPARROW)

This document describes how users move through the app.

---

## 1. First-Time User Flow

1. User opens the app
2. Splash Screen appears
3. User is welcomed on Home Screen
4. User is shown:
   - Daily fun fact or joke
   - Friends of Nature highlight
5. User opens Navigation Drawer to explore

---

## 2. Returning User Flow

1. App opens
2. Splash Screen
3. Home Screen shows:
   - Updated daily content
   - Quick access cards
4. User continues where they left off

---

## 3. Animal Exploration Flow

1. Home → Animals
2. Browse or search animals
3. Select animal
4. View Animal Detail
5. Save to favorites or return

---

## 4. Bird Exploration Flow

1. Home or Drawer → Birds
2. Browse or search birds
3. Select bird
4. View Bird Detail
5. Play bird call or return

---

## 5. Parks Exploration Flow

1. Home or Drawer → Parks
2. View nearby parks
3. Select park
4. View Park Detail
5. Explore map or wildlife highlights

---

## 6. Friends of Nature Flow

1. Home or Drawer → Friends of Nature
2. View daily content
3. Read joke, tip, or conservation message
4. Return to Home or explore related content

---

## 7. Quiz Flow

1. Drawer → Quiz & Games
2. Select quiz type
3. Answer questions
4. View results
5. Return or retry

---

## 8. RAFIKI Chatbot Flow

1. Drawer → RAFIKI Chatbot
2. Ask a question
3. Receive response
4. Continue chat or exit

---

## 9. Safari Journal Flow

1. Drawer → Safari Journal
2. Add or view entries
3. Save content offline
4. Return to Home

---

## 10. Profile & Settings Flow

1. Drawer → Profile
2. View or edit profile
3. Drawer → Settings
4. Adjust preferences
5. Return

---

## 11. Guide Flow (If User Is a Guide)

1. Guide opens app
2. Access guide tools
3. Share features with nearby guests
4. Monitor sessions and limits
5. End session
